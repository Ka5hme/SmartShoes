# SmartShoes
A shoe that creates a real-time contour map of your foot while in use.
# Parts
- Raspberry Pi Pico WH
- Force Sensitive Resistors (FSR)
- Resistors
- Analog to Digital Converter (ADC)
- Power Supply/Battery
